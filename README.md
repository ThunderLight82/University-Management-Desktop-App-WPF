# "University Management" (Desktop) Project

The Desktop Application project seamlessly organizes university affairs, from personnel and courses to groups and students, all within a user-friendly interface.

## Tech Stack

- WPF
- XAML
- EF Core
- MSSQL
- CsvHelper
- DocX
- Moq
- xUnit
